📊 Hospital Data Analysis using SQL & Excel
🔍 Overview

This project focuses on analyzing hospital data using SQL in PostgreSQL and visualizing insights using Microsoft Excel. The dataset includes information about hospitals, departments, patient count, and medical expenses to derive meaningful business insights.

🎯 Objectives
Analyze patient distribution across departments and cities
Identify hospitals with the highest medical expenses
Track monthly trends in healthcare costs
Perform real-world data analysis using SQL
🛠 Tools & Technologies
PostgreSQL (pgAdmin)
Microsoft Excel
SQL (Structured Query Language)
📈 Key Insights
Urology department has the highest number of patients
Jaipur and Ahmedabad have the highest patient load
Medical expenses fluctuate across different months
Some hospitals have higher average doctor availability
